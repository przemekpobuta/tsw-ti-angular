Przedmiot obieralny

Technologie programistyczne usług internetowych

Wykład

  1. Technologia serwlet XSQL i język XSQL,
  2. Przekształcanie i prezentacja dokumentów XML w technologii XSQL,
  3. Usługi sieciowe (Web Services) w technologii XSQL,
  4. Tworzenie aplikacji internetowych w środowisku .NET,
  5. Usługi sieciowe w technologii .NET,
  6. Bezpieczeństwo aplikacji internetowych,
  7. Dystrybucja i instalacja aplikacji.

W ramach przedmiotu zostanie dokonany przegląd nowych technologii, które mogą znaleźć zastosowanie we współczesnych aplikacjach. Servlet XSQL to technologia firmy Oracle, która pozwala na proste połączenie danych zapisanych w formacie XML z relacyjną bazą danych. Pierwszy wykład jest wprowadzeniem do technologii. Omawia strukturę serwletu i język XSQL, służący do budowy stron XSQL. Na przykładach pokazuje przeprowadzanie operacji CRUD na danych w postaci XML. Wykład drugi poświęcony jest zagadnieniom przekształcania danych zapisanych w języku XML. Dane przesłane do strony XSQL z przeglądarki klienta mogą wymagać pewnych przekształceń przed wstawieniem ich do bazy danych. Podobnie, klientowi nie prezentujemy danych w postaci XML, tylko przekształcone do języka HTML. Przekształcanie danych odbywa się przy użyciu języka XSLT. W ramach przedmiotu omawiane są wybrane cechy środowiska programistycznego Visual Studio .NET. Na przykładzie języka C# przedstawiany jest sposób tworzenia aplikacji internetowych, wykorzystujących bazy danych. Kolejne wykłady poruszają zagadnienie tworzenia usług sieciowych w technologii XSQL. Usługa sieciowa to komponent programowy, zaprojektowany w celu wsparcia przetwarzania danych i komunikacji pomiędzy automatami połączonymi siecią. Omawiane jest także wsparcie dla usług sieciowych w środowisku programistycznym Visual Studio .NET. W ramach przedmiotu przedstawiane są wybrane aspekty bezpieczeństwa internetowych aplikacji bazodanowych. Podawane są sposoby unikania zagrożeń oraz odpowiednie techniki programistyczne. Na zakończenie omawiane są zagadnienia dystrybucji i instalacji gotowego oprogramowania za pomocą narzędzi Visual Studio .NET.

Laboratorium

  1. Podstawowe operacje CRUD (Create, Read, Update, Delete) na danych. Konfiguracja środowiska i tworzenie podstawowych stron XSQL, służących do prezentacji danych z bazy danych, wstawianie, aktualizacja i usuwanie danych,
  2. Przekazywanie danych (parametrów) pomiędzy poszczególnymi stronami XSQL,
  3. Prezentacja danych XML, opracowanie arkuszy transformacji w języku XSLT służących do przekształcania danych z języka XML na HTML w technologii XSQL,
  4. Wprowadzanie danych do formularzy HTML, techniki stosowane podczas przesyłania danych z przeglądarki klienta do strony XSQL,
  5. Tworzenie aplikacji internetowych w języku C#. Przykład budowy internetowej aplikacji bazodanowej (MS SQL Server / MS Access) w środowisku Visual Studio .NET,
  6. Usługi Web Services w .NET , korzystanie z usług sieciowych w programach .NET oraz tworzenie własnych usług,
  7. Tworzenie wersji instalacyjnych oprogramowania, przegląd i zastosowanie narzędzi do dystrybuowania aplikacji dostępnych w środowisku Visual Studio .NET.
